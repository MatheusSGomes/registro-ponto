# Registro de Ponto
- Linguagem de programação PHP para o backend
- Framework Laravel no backend
- Banco de dados PostgreSQL
- Testes unitários.
- Frontend em Vue JS
- GIT para versionamento da aplicação.
- Docker para deploy da aplicação.
- Implementação UUID
